# E-Commerce User Analytics Project

This project analyzes user behavior on an ecommerce platform using SQL.

## Tools Used
- MySQL

## What I analyzed
- Users by country and language
- Buyers and sellers by country
- Male vs female user behavior
- Android vs iOS platform usage
- Inactive users (not logged in for 100+ days)
- Wishlist and social like behavior

## How to run
1. Load users_data table into your database
2. Open sql/queries.sql
3. Run queries
